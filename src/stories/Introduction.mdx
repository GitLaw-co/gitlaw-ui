# GitLaw UI Component Library

Welcome to the **GitLaw UI** component library, built with React and Tailwind CSS. This Storybook showcases the design system extracted from the GitLaw Figma library.

## Components

### Button
A versatile button component with multiple variants (primary, secondary, outline, ghost, destructive, disabled) and sizes (xs, s, m, l, xl).

### Input
Text input component with support for labels, icons, and multiple sizes.

### Avatar
User avatar component supporting images, initials, icons, and empty states in multiple sizes.

### Badge
Notification badge component with color variants and optional count display.

### Tag
Label/tag component for categorization with icon and close button support.

### Tab
Navigation tab component with badge support and selection states.

### Switch
Toggle switch component for boolean settings.

### ListItem
List item component with multiple left element types (icon, avatar, clause, select, mini-users), secondary text, and selection states.

### Select
Dropdown select component with search functionality, multiple sizes, validation states, and label positioning options.

### TextField
Multi-line text area component with label support, multiple sizes, and validation states.

### Sidebar
Navigation sidebar component with multiple variants and states:
- **Variants**: `landing` (dark purple background) and `inner` (light purple background)
- **States**: `collapsed` (icons only, 64px) and `expanded` (full content, 288px)
- **User states**: `signed-in` (with profile and chat history) and `signed-out` (marketing menu)
- Includes logo, menu items, chat history (signed-in), and user profile section

### Tooltip
Hover tooltip component with multiple sizes and color types:
- **Sizes**: `s` (12px text), `m` (14px text), `l` (16px text)
- **Types**: `neutral` (white background, dark text) and `purple` (purple background, white text)
- **Positions**: `top`, `right`, `bottom`, `left`

### Card
Content card component for displaying files, templates, and folders:
- **Types**: `file`, `template` (same visual format), `folder`
- **File/Template cards**: Display owner avatar, title, visibility status, description
- **Folder cards**: Display folder header with file count and list of files inside
- **Features**: Star/favorite toggle, menu button, hover overlay with action button
- **Responsive**: Works at various breakpoints (xxs: 288px, xs: 204px, sm: 192px, md: 239px, lg: 242px, xl: 215px, 2xl: 202px)
