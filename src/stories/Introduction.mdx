# GitLaw UI

An **experimental** React component library built with Tailwind CSS, generated using Claude from the GitLaw Figma design system.

> **Note:** This is a proof-of-concept implementation created with AI assistance. It demonstrates how Claude can translate Figma designs into functional React components. Not intended for production use (yet!).

## About This Project

This Storybook showcases 38 components, 1,475 icons, and 100+ illustrations — all generated through an experimental workflow using Claude to interpret Figma designs. Use the sidebar to explore components interactively.

### How to Navigate

| Section | What You'll Find |
|---------|------------------|
| **Components** | Interactive component demos with controls |
| **Foundations** | Colors, typography, spacing, icons |
| **Docs** | Component API documentation |

> **Tip:** Use the **Controls** panel at the bottom of each story to experiment with props in real-time.

## Quick Start

```tsx
import { Button, Input, Avatar, Icon } from './components';
import { colors } from './specs';

// Primary button
<Button variant="primary" size="m">Get Started</Button>

// Input with label
<Input label="Email" placeholder="you@example.com" showLabel />

// User avatar
<Avatar type="initials" initials="JD" size="M" />

// Icon
<Icon name="check" className="size-6" color={colors.iconPrimary} />
```
